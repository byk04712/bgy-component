# Vue2.x + Vite2.x + TypeScript 基于 Ant Design for Vue 封装通用组件库

[![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)
### 目录结构
```
├── build # 编译脚本
├── dist # 打包后的代码
├── docs # 文档目录
├── coverage # 覆盖率报告（暂无）
├── examples # 代码范例
├── node_modules
├── packages # 组件代码
├── scripts # 脚本 发布、提交信息检查
├── src # 通⽤代码
├── test # 测试（暂无）
└── types # TS类型定义
```
